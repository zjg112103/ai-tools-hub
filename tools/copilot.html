📖 怎么用？

  1. 确保有 GitHub 账号：没有的话去 github.com 注册
  2. 安装 VS Code：如果还没装的话，下载 VS Code
  3. 安装 Copilot 插件：在 VS Code 插件市场搜索"GitHub Copilot"，点击安装
  4. 登录 GitHub：安装后会提示登录 GitHub 账号授权
  5. 开始写代码：正常写代码，Copilot 会自动给出建议，按 Tab 接受

也支持 JetBrains、Neovim、Visual Studio 等其他编辑器。

❓ 常见问题

Copilot 写的代码有版权问题吗？

GitHub 表示 Copilot 不会逐字复制训练数据中的代码，并提供了版权免责条款。企业版用户可以额外开启"公共代码过滤"，避免生成与开源代码太相似的片段。如果你在商业项目中使用，建议了解清楚 GitHub 的服务条款。

学生怎么免费申请？

申请 GitHub 学生认证（GitHub Student Developer Pack），通过后 Copilot 免费用。需要用学校邮箱（.edu.cn 等）或提供学生证明。中国大陆很多大学的邮箱都支持。

Copilot 国内能用吗？

能。GitHub 在国内可以正常访问（虽然有时不稳定），VS Code 插件安装后正常使用不需要翻墙。个别地区可能偶尔连接慢，但大多数时候没问题。

和 Codeium 比怎么样？

Codeium 是免费的 AI 编程插件，补全质量不如 Copilot，但完全免费且不限次数。如果你不想花钱、轻度使用，Codeium 是很好的免费替代。如果你是专业开发者，Copilot 的补全质量还是更好的。
