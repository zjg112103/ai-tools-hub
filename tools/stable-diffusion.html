📖 怎么安装？两种方式

方式一：WebUI（推荐新手）

  1. 安装 Python 和 Git：确保电脑装了 Python 3.10+ 和 Git
  2. 下载 WebUI：搜索"Stable Diffusion WebUI"，从 GitHub 下载 Automatic1111 版本
  3. 运行启动脚本：双击 run.bat（Windows），自动安装依赖和下载模型
  4. 打开浏览器：启动完成后浏览器会自动打开 http://127.0.0.1:7860
  5. 开始画图：在 txt2img 页面输入 Prompt，点击 Generate

方式二：ComfyUI（推荐进阶）

ComfyUI 用节点式工作流，学习曲线陡但可控性更强。适合已经熟悉 SD 基础、想精细控制出图流程的用户。许多最新的模型和工作流（如 Flux）优先支持 ComfyUI。

❓ 常见问题

不会编程能用 Stable Diffusion 吗？

能。安装 WebUI 后界面和普通网站差不多，不需要写代码。但安装过程需要一些基础操作（安装 Python、运行命令行）。如果这步都嫌麻烦，推荐用 Midjourney 或者国产的即梦 AI、LiblibAI（网页版，不用本地安装）。

模型去哪里下载？

主要在 Civitai（最大的 SD 模型社区）和 Hugging Face 上下载。Civitai 上有各种风格的大模型和 LoRA（微调模型），可以按分类和评分筛选。下载后放到 WebUI 的 models/Stable-diffusion 文件夹即可。

SD XL 和 SD 1.5 有什么区别？

SDXL 是更新的模型，画质更好、理解能力更强，但需要更多显存（建议 12GB+）。SD 1.5 是经典版本，社区模型最多、兼容性最好，8GB 显存就能跑。建议新手先从 SDXL 开始（画质直观更好），资源不够再用 SD 1.5。

Flux 模型怎么样？

Flux 是 Black Forest Labs（Stable Diffusion 原班人马创立）推出的新模型，画质和文字渲染能力大幅提升，目前是开源绘画模型中的顶级水平。但需要更好的硬件配置，建议 16GB+ 显存。
