📖 怎么开始？

  1. 下载安装：访问 cursor.com，下载对应系统的安装包（Windows/Mac/Linux 都支持）
  2. 导入 VS Code 配置：首次打开会问你是否导入 VS Code 的插件和设置，建议选"是"
  3. 打开项目：用 Cursor 打开你现有的项目文件夹
  4. 开始用 AI：按 Ctrl+L（Mac: Cmd+L）打开 AI 对话框，直接用中文描述你的需求
  5. 试试 Composer：按 Ctrl+I（Mac: Cmd+I）打开 Composer 模式，AI 可以同时修改多个文件

💡 实用技巧

  • @codebase：在对话框输入 @codebase，让 AI 搜索整个项目的相关代码
  • @docs：输入 @docs + 技术名称（如 @docs React），AI 会查阅官方文档再回答
  • 选中代码 + Cmd+K：选中一段代码，按快捷键，直接告诉 AI 怎么修改
  • Composer 多文件编辑：大功能开发时用 Composer 模式，AI 会同时修改多个文件并保持一致性
  • .cursorrules 文件：在项目根目录创建这个文件，写上你的编码规范，AI 会自动遵守

❓ 常见问题

不会编程的人能用 Cursor 吗？

能。Cursor 降低了编程门槛，你只需要用自然语言描述想要什么。但完全零基础的话，AI 生成的代码你可能不知道怎么调试。建议至少了解基本的编程概念（变量、函数、循环）再使用。

Cursor 会让程序员失业吗？

不会，但会改变工作方式。AI 能帮你写代码，但架构设计、业务理解、代码审查这些还是需要人。会用 AI 工具的程序员效率会远高于不用的。

Cursor 国内能用吗？

能。Cursor 编辑器可以直接下载使用，国内不需要翻墙。AI 功能需要联网调用模型 API，但 Cursor 自己的服务器可以做中转。实际使用中，国内访问基本流畅。

和 TRAE 比怎么样？

TRAE 是字节跳动推出的 AI 编程 IDE，也是免费的，中文场景体验不错。区别是 Cursor 生态更成熟、模型选择更多，TRAE 则完全免费且对中文用户更友好。如果你不想花钱，可以两个都试试看哪个顺手。
