📖 怎么用？

  1. 打开官网：访问 claude.ai
  2. 注册账号：用邮箱或 Google 账号注册。注意：需要科学上网
  3. 开始对话：直接输入问题，和 ChatGPT 用法基本一样
  4. 上传文件：点输入框左侧的附件按钮，可以上传 PDF、Word、图片等
  5. 使用 Projects：Pro 用户可以创建项目，上传多个文件让 AI 持续参考

💡 什么是 Artifacts？

Artifacts 是 Claude 的特色功能。当你让 Claude 生成代码、表格、图表、SVG 图形等内容时，它会在右侧单独展示一个可交互的预览窗口，而不是混在对话里。你可以直接看到运行效果、复制代码、甚至在此基础上修改。非常适合：

  • 生成网页代码 → 直接预览效果
  • 生成图表/流程图 → 可视化查看
  • 生成文档/表格 → 格式化展示

❓ 常见问题

Claude 国内能用吗？

claude.ai 需要科学上网。另外注册时可能需要海外手机号验证。如果你不想折腾翻墙，DeepSeek 和智谱清言是更好的国内替代选择。

Claude 写文章真的比 ChatGPT 好吗？

大多数写作者的共识是：是的。Claude 写的中文更自然、更有"人味"，不太像 AI 生成的。ChatGPT 写的内容有时会有明显的 AI 味（喜欢用"总的来说"、"值得注意的是"这类套话）。但具体好多少取决于你的使用场景，建议两个都试试，自己感受差异。

Claude Opus 和 Sonnet 有什么区别？

Opus 是 Claude 系列的最强模型，推理能力和复杂任务表现最好，但只有 Pro 用户能用。Sonnet 是中等模型，免费用户就用这个，日常写作、对话已经足够强了。简单说：要最强选 Opus（付费），日常用 Sonnet（免费）就行。

Claude 能画画吗？

不能生成图片。Claude 目前只支持文字对话、文件分析和图片理解（看图说话）。如果你需要生成图片，推荐 Midjourney 或通义万相。
