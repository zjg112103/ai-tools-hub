📖 怎么用？

  1. 开通 ChatGPT Plus：DALL·E 3 需要订阅 ChatGPT Plus（$20/月）。部分免费用户也有有限的 DALL·E 额度
  2. 在 ChatGPT 里说：直接输入"帮我画一张……"或"generate an image of..."
  3. 等 10-30 秒：AI 生成图片，不满意可以继续对话修改
  4. 下载：点击图片可以下载高清版本

没有 Plus？可以用微软的 Bing Image Creator（免费，也用的 DALL·E 3），访问 bing.com/images/create 即可。

❓ 常见问题

DALL·E 3 一定要付费吗？

最方便的使用方式是 ChatGPT Plus（$20/月）。但微软的 Bing Image Creator 也用 DALL·E 3 且免费（每天有额度），只是功能没有 ChatGPT 里那么完整。国内用户可以用 Bing 访问。

DALL·E 3 和 DALL·E 2 有什么区别？

DALL·E 3 是更新版本，画面质量更好、文字渲染更准确、理解能力更强。现在 OpenAI 默认使用 DALL·E 3，DALL·E 2 已经很少用了。

生成的图片有版权吗？

OpenAI 表示付费用户生成的图片归用户使用，但 AI 生成的图片在大多数国家不能获得版权保护。建议用作素材和创意参考，不要直接作为核心商业资产。
